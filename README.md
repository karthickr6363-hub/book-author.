# T. Abernathy - Author Website

A complete, modern website for an author/author services business with multiple pages and features.

## Project Structure

```
book author/
├── index.html              # Home Page 1 (General Author Landing)
├── index2.html            # Home Page 2 (Agency/SaaS Style)
├── about.html             # About Us Page
├── services.html          # Services Page
├── service-details.html    # Service Details Page
├── books.html             # Books Page
├── book-details.html      # Book Details Page
├── blog.html              # Blog Page
├── blog-details.html      # Blog Details Page
├── contact.html           # Contact Us Page
├── pricing.html           # Pricing Page
├── login.html             # Login Page
├── register.html           # Register Page
├── admin-dashboard.html    # Admin Dashboard
├── coming-soon.html        # Coming Soon Page
├── css/
│   ├── style.css          # Main stylesheet
│   ├── home.css           # Home page specific styles
│   ├── agency.css         # Agency style home page styles
│   └── admin.css          # Admin dashboard styles
└── js/
    ├── main.js            # Main JavaScript file
    └── services.js        # Services page JavaScript
```

## Features

### Home Page 1 (index.html)
- Hero section with overlay content box
- Featured books slider
- About the author section
- Services showcase
- Testimonials
- Blog preview
- Newsletter signup
- Footer

### Home Page 2 (index2.html)
- Hero with metrics
- Service highlights
- Process timeline
- Pricing preview
- Client logos
- Call to action

### Other Pages
- **About Us**: Biography, mission, vision, timeline, awards, testimonials, team
- **Services**: Grid/list view toggle, service cards with details
- **Service Details**: Detailed description, benefits, pricing table, FAQs
- **Books**: Book grid with filters, search, ratings, buy links
- **Book Details**: Synopsis, author notes, chapters preview, reviews
- **Blog**: Blog listing with filters, search, pagination
- **Blog Details**: Full article, author box, tags, related posts, sidebar
- **Contact**: Contact form, Google Map, contact info, social links
- **Pricing**: Author services pricing and subscription plans
- **Login/Register**: User authentication pages
- **Admin Dashboard**: Analytics, user management, orders, messages, blog management
- **Coming Soon**: Countdown timer and email subscription

## Design Theme

- **Colors**: Dark blue (#1a3a5f), beige/cream (#f5f1e8), accent brown (#8b6f47)
- **Typography**: Georgia serif font for elegant, literary feel
- **Style**: Modern, clean, library/bookstore theme
- **Responsive**: Mobile-friendly design

## JavaScript Features

- Mobile menu toggle
- Book slider functionality
- Newsletter form validation
- Accordion (FAQs)
- Filter functionality (books, blog)
- Search functionality
- Shopping cart (localStorage)
- Form validation
- Smooth scrolling

## Usage

1. Open `index.html` in a web browser to view the home page
2. Navigate through pages using the header navigation
3. All pages are fully functional with interactive elements
4. Shopping cart uses localStorage for persistence

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design works on desktop, tablet, and mobile devices

## Notes

- Images use placeholder URLs - replace with actual images
- Google Maps embed uses placeholder - update with actual location
- Form submissions are simulated (no backend)
- Shopping cart data persists in browser localStorage
